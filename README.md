# RHCSA Master Hub

An advanced RHCSA study website built with HTML, CSS, and JavaScript.

## Overview

This project contains a modern, interactive RHCSA training portal with:

- Advanced responsive UI layout
- Sidebar chapter navigation
- Search across sections and commands
- Progress tracking using localStorage
- Theme toggle (dark/light)
- Code block syntax highlighting and copy buttons
- Keyboard shortcuts and productivity features

## Run Locally

1. Clone the repository.
2. Open the project folder in VS Code.
3. Open website/index.html in a browser.
