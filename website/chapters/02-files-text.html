File Operations and Text Processing

Operate safely on files and directories, then extract exact information fast using grep, sort, and stream tools.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Create/move/copy/remove files and directories confidently
  • Use wildcard, brace, variable, and command expansion
  • Use head, tail, cat, and less for controlled viewing
  • Use grep and sort for filtering and ordering output
Advanced Execution
  • Build precise grep patterns with anchors and whole-word matching
  • Combine pipelines to produce one-line proof output
  • Use safe deletion habits in recursive structures
  • Optimize exam tasks with expansion shortcuts
Core Command Workflow
mkdir -p app/{conf,logs,backup}
cp -r app app-copy
mv app-copy app-final
grep -r "PermitRootLogin" /etc/ssh
grep "bash$" /etc/passwd
sort -u users.txt
tail -f /var/log/messages
Validation and Troubleshooting
  • find app-final -maxdepth 2 -type d
  • wc -l users.txt
  • grep -n "root" /etc/passwd
