Networking with nmcli and ip tools

Configure persistent network profiles and troubleshoot connectivity from interface state to DNS resolution.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Inspect adapters and addresses with ip
  • Configure static IPv4 with nmcli
  • Bring connections up/down safely
  • Verify route and gateway behavior
Advanced Execution
  • Troubleshoot layered failures: link, IP, route, DNS
  • Use hostname and /etc/hosts strategically
  • Validate listening sockets and active connections
  • Prepare quick recovery sequence under exam time limits
Core Command Workflow
nmcli con show
nmcli con mod "Wired connection 1" ipv4.addresses 192.168.56.20/24
nmcli con mod "Wired connection 1" ipv4.gateway 192.168.56.1
nmcli con mod "Wired connection 1" ipv4.method manual
nmcli con up "Wired connection 1"
ip a
ip r
Validation and Troubleshooting
  • ping -c 3 8.8.8.8
  • ss -ta
