Shell Scripting and Automation

Build robust shell scripts for repeatable administration tasks and structured exam automation.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Use shebang and executable permissions
  • Use variables, loops, and conditionals
  • Capture command output and redirect logs
  • Write scripts that fail fast with strict mode
Advanced Execution
  • Combine find with -exec for bulk operations
  • Build reusable service health checks
  • Handle errors and edge cases predictably
  • Produce audit artifacts for examiner validation
Core Command Workflow
#!/usr/bin/env bash
set -euo pipefail
for s in sshd chronyd firewalld; do systemctl is-active --quiet "$s" || echo "$s down"; done
find /usr -perm /g+s -size +30k -size -45k > /root/usr-files
chmod +x audit.sh
./audit.sh
Validation and Troubleshooting
  • bash -n audit.sh
  • head -n 5 /root/usr-files
  • echo $?
