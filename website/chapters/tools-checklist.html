Final Validation and Exam Checklist

Use a strict final verification sweep to ensure no hidden marks are lost in RHCSA tasks.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Validate identity and sudo permissions
  • Validate mounts, swap, and fstab syntax
  • Validate services active/enabled state
  • Validate network and hostname reachability
Advanced Execution
  • Run SELinux/firewalld checks for service access
  • Review recent logs for silent failures
  • Re-check typo-prone files: fstab, sudoers, service units
  • Use command evidence files when exam asks for proof
Core Command Workflow
id user
sudo -l
mount -a
swapon --show
systemctl is-active sshd chronyd firewalld
ip a
sestatus
firewall-cmd --list-all
Validation and Troubleshooting
  • journalctl -p err..alert --since "-30 min"
  • grep -v "^#" /etc/fstab
