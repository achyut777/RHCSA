Logs, Journald, rsyslog, and Time Sync

Collect, filter, and analyze logs while maintaining accurate system time with chrony and timedatectl.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Use journalctl for service and boot diagnostics
  • Use /var/log/messages and /var/log/secure effectively
  • Set timezone with timedatectl
  • Enable NTP synchronization via chronyd
Advanced Execution
  • Correlate multi-service incidents through timestamp alignment
  • Tune chrony sources and validate sync state
  • Use priority and time-range filters in journalctl
  • Build quick triage flow for auth/network/service failures
Core Command Workflow
journalctl -xe
journalctl -u sshd --since "-15 min"
tail -f /var/log/secure
timedatectl
timedatectl set-timezone America/Jamaica
chronyc sources
Validation and Troubleshooting
  • systemctl status chronyd
  • timedatectl status
  • grep -i chrony /var/log/messages
