CH8

LVM and Swap Lifecycle

Build flexible logical storage with LVM and configure swap persistently for memory management.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Create PV, VG, and LV in correct order
  • Apply filesystem and persistent mounts on LV
  • Configure swap partition with mkswap and swapon
  • Verify state using pvs/vgs/lvs and swapon
Advanced Execution
  • Extend VG with additional PV when free extents are exhausted
  • Grow XFS online with xfs_growfs
  • Reduce only EXT filesystems with careful offline workflow
  • Tear down LVM resources in proper reverse order
Core Command Workflow
pvcreate /dev/vdc1
vgcreate vgdata /dev/vdc1
lvcreate -n lvapp -L 4G vgdata
mkfs.xfs /dev/vgdata/lvapp
lvextend -L +1G /dev/vgdata/lvapp
xfs_growfs /dev/vgdata/lvapp
swapon --show
