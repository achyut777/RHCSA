Scheduling: at, cron, anacron

Automate one-time and recurring jobs with proper ownership, syntax, and persistence controls.

Back To Handbook Open Simulator
From Scratch (Foundation)
  • Schedule one-time tasks with at and manage queue
  • Create recurring tasks with crontab syntax
  • Differentiate user crontab vs /etc/crontab format
  • Use cron directories for periodic scripts
Advanced Execution
  • Control cron usage via cron.allow/cron.deny
  • Use anacron for non-24x7 systems
  • Design idempotent scheduled scripts
  • Audit scheduled execution via logs
Core Command Workflow
at now + 5 minutes
atq
atrm 1
crontab -e
crontab -l
cat /etc/crontab
ls /etc/cron.daily
Validation and Troubleshooting
  • tail -f /var/log/cron
  • journalctl -u crond --since "-30 min"
