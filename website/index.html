Chapter 1

User and Authentication Management

Create accounts with controlled shells, enforce aging, and delegate least-privilege access using sudo.

Core Commands
useradd -m analyst
Create user with home directory.
passwd analyst
Set or reset account password.
chage -M 90 analyst
Set max password age in days.
usermod -aG wheel analyst
Append user to admin group.
Exam Workflow
  1. 1
    Create users first and set required shell and groups in one pass.
  2. 2
    Apply password policy with chage for expiry and warning windows.
  3. 3
    Verify quickly with id, getent passwd, and sudo -l.

Chapter 10

Bash Scripting Essentials

Automate repetitive checks with shell scripts that are readable and safe.

Service Health Script
#!/usr/bin/env bash
set -euo pipefail
for svc in sshd chronyd firewalld; do
  if systemctl is-active --quiet "$svc"; then
    echo "OK: $svc"
  else
    echo "FAIL: $svc"
  fi
done
